Ce module complémentaire décrit une position de jeu d'échecs a partir de un code FEN.

La notation FEN (Forsyth–Edwards) est une méthode pour représenter une position dans les échecs.

  • Sélectionner du texte avec un code FEN valide
  • Appuyer sur NVDA+Contrôle+F8 pour entendre une description de la position
  • Appuyer sur NVDA+Maj+F8 pour écouter la position et le copier dans le presse-papiers
  • Appuyer sur NVDA+Contrôle+Maj+F8 pour changer la langue de la notation

Note sur la langue de la notation : la langue de la notation désigne les abréviations des pièces utilisées dans le code FEN. Sont des abréviations couramment utilisées en anglais, mais il est parfois aussi possible d'utiliser la langue locale. Le module complémentaire utilise par défaut la notation anglaise mais vous pouvez éventuellement changer à la notation française.
