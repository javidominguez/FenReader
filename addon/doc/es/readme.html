Este complemento describe una posición de ajedrez a partir de un código FEN.

La notación FEN (Forsyth–Edwards) es un método para representar una posición en ajedrez.

  • Selecciona texto con un código FEN válido
  • Pulsa NVDA+Control+F8 para escuchar una descripción de la posición
  • Pulsa NVDA+Shift+F8 para escuchar la posición y copiarla al portapapeles
  • Pulsa NVDA+Control+Shift+F8 para cambiar el idioma de notación

Nota sobre el idioma de notación: El idioma de notación se refiere a las abreviaturas de las piezas utilizadas en el código FEN. Se suelen utilizar las abreviaturas en inglés, pero a veces también es posible usar las del idioma local. El complemento usa por defecto la notación inglesa pero opcionalmente se puede cambiar a la notación española.
